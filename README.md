# BotOps

[![Greenkeeper badge](https://badges.greenkeeper.io/juzibot/botops.svg)](https://greenkeeper.io/)

Using Bot to automate everything!

## Plugins

`src/plugins`目录

- scp: 获取SCP基金会某个保存物品的说明
- weather: 天气推送和查询
- tbp: 腾讯对话平台闲聊
- version: 获取版本号
- zhihu-follower: 查询知乎粉丝数
